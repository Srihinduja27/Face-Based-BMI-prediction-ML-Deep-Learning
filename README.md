# Face-Based BMI Prediction Using Deep Learning

## Overview
This project predicts Body Mass Index (BMI) from facial images using MobileNetV2 and Deep Learning techniques.

## Features
- Facial image preprocessing using OpenCV
- Transfer Learning with MobileNetV2
- BMI prediction from facial images
- BMI categorization
- Model evaluation and visualization

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
UTKFace Dataset
